Recycle In Peace

Recycle in Peace is a Webgl application that creates environmental a wareness through an interactive experience about disposing batteries in the correct bin.

3D Modelling / HTML / CSS / Three.js

Challange

“How might we create environmental awareness in an interactive form”

Tools

  • Cinema4D
  • HTML/CSS
  • javascript
  • Three.js

    • Problem

    The lack of awareness about protecting environment is harmful for any living being

    Concept

    Recycle in peace is a fun experience that makes the learning easy. It is visually interesting due to the graveyard theme with spooky effects.

    Target Audience

    Pre-Teens

    Ideation

    The issue that we wanted to deal with was to create an interactive experience for everyone about protecting the environment. The initial idea was to create awareness about melting glaciers and increase in heat but eventually I learnt that a lot of small problems creates one huge problem. Hence, I thought of focusing on one tiny problem that could be harmful for the environment and living beings if not taken care of.

    Pain Points

    - Lack of awareness
    - Information Overload , Difficulty understanding terminology

    Solution

    - Creating awareness about discarding batteries in the correct bin and the cons of throwing the batteries in any other bins
    - It is an interactive experience that allows user to gain knowledge in a playful way
    - Ability to learn about the use of different garbage bins and what should be discarded in those

    Experience

    Users would be able to hit the start button in order to learn more. Later would be given 3 bins and asked where the battery should be discarded and upon clicking all 3 bins they would find out none of the bins were correct and the screen would then show the right bin and also provide knowledge and the battery would be discarded into it.

    3D Model

    The 3D Model is fitting to a graveyard. The uneven surface is meant to create an unease. In addition, their is a layer of fog that follows that spooky theme. Because the main focus is on garbage bins, the only colors that stand different is the bins themselves.

    Assets

    -Spooky Fence
    - Aluminium Gate
    -A Bin
    - A Battery Google Maps Api - Utilizing the Google Maps Api in order to track the location of the user in order to display results by location

    Navigation

    1. It starts by hitting the “Start” Button
    2. A battery would appear with a question of where to dispose it
    3. It will show 3 bins and the user would be asked to click the correct bin
    4. While they click the bin, they would discover that those bins aren’t correct and are used for different purposes
    5. Later the users would be taken to the correct bin and it will also talk about why it is important to dispose the batteries in the correct bin
